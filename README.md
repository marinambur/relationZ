# Relationz
**Описание проекта**

 Одностраничный лендинг с элементами анимации.

### Технологии, использующиеся в проекте
* Адаптивная верстка
* Резиновая верстка
* Медиазапросы
* Анимация, кейфреймы
* Флексбоксы
* Интерактивные элементы (ссылки, кнопки)
